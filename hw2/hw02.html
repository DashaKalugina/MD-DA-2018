Функция apply() применяет функцию is.na() ко всем элементам таблицы (т.к. параметру margin задается значение c(1,2), обозначающее применение функции и строкам, и столбцам). Функция is.na() возвращает FALSE, если данный элемент датафрейма содержит значение, и TRUE в том случае, если элемент указывает на пропущенное значение NA (Not Available). Т.е. в результате выполнения команды apply(data,c(1,2), is.na) будет получена матрица логических значений.

Функция colSums() подсчитывает количество значений TRUE (пропущенных значений) для каждого столбца, т.е. сколько значений пропущено по каждой переменной.

  • Функция na.omit() принимает датафрейм и возвращает новый датафрейм, игнорируя строки, содержащие значение NA. Используйте эту функцию для удаления строк с неполными данными.
data <- na.omit(data)

Корреляция

Колонка COUNTYFP содержит числовой код округа внутри штата. Нас интересуют Butte County (округ 7 в Калифорнии (штат 6)), Santa Clara (округ 85 в Калифорнии (штат 6)) и York County (округ 133 в Пенсильвании (штат 42)).

  • Объясните, что делает приведённый в конце задания код и как именно он это делает.
acc <- c()
for (tract in 1:nrow(data)) {
  if (data$STATEFP[tract] == 6) {
    if (data$COUNTYFP[tract] == 1) {
      acc <- c(acc, tract)
    }
  }
}
accmv <- c()
for (tract in acc) {
  accmv <- c(accmv, data[tract,10])
}
median(accmv)
## [1] 474050

В первом цикле приведенного кода просматриваются все строки датафрейма data и в переменную acc добавляются номера строк, соответствующие округу 1 в штате 6 (Калифорния). Во втором цикле в переменную accmv для каждого значения номера строки tract, сохраненного в вектор acc, записывается значение десятого столбца (т.е. значение медианы стоимости домовладений для каждой зоны переписи в пределах округа 1 штата Калифорния). В конце находится медиана полученного вектора accmv.
